# Kana Memory | 在線日語學習工具

一個優雅的日語學習 Web 應用，提供假名、單詞、句子三大學習模式，支持快捷鍵導航與全鍵盤操作，助你系統化掌握日語。

🌐 **在線體驗**: [kana.yahaha.net](https://kana.yahaha.net/)

## ✨ 主要功能

### 📚 三大學習模式

- **假名學習** - 完整支持清音 46 個、濁音 25 個、拗音 33 個（共 104 個）
- **單詞學習** - 按假名行分類學習日語單詞，支持拗音單詞，智能篩選
- **句子學習** - 場景化句子練習，涵蓋日常對話、旅遊、商務等多個場景

### 🎯 學習功能

- **多種顯示模式** - 混合 / 平假名 / 片假名 / 羅馬音 / 互換
- **雙重練習模式** - 學習模式（自動提示）/ 記憶模式（手動提示）
- **智能 TTS 發音** - 基於 Web Speech API 的標準日語朗讀
- **自動發音選項** - 學習模式下可開啟自動朗讀
- **假名選擇器** - 可視化選擇清音、濁音、拗音範圍
- **鍵盤快捷鍵** - 全鍵盤操作支持，含 Gmail 風格跳轉（G+H/K/W/P）與 Enter 開始
- **主題模式** - 亮色 / 暗色 / 跟隨系統
- **響應式設計** - 完美適配手機、平板、桌面
- **數據持久化** - 自動保存學習進度和偏好設置

## 🚀 快速开始

### 本地运行

```bash
# 克隆项目
git clone https://github.com/dofy/KanaSyllabaryMemory.git
cd KanaSyllabaryMemory

# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 浏览器访问 http://localhost:3000
```

### 构建部署

```bash
# 构建生产版本
pnpm build

# 预览构建结果
pnpm preview
```

## ⌨️ 鍵盤快捷鍵

| 快捷鍵                | 功能                     |
| --------------------- | ------------------------ |
| `S`                   | 開關設置                 |
| `?`                   | 開關幫助                 |
| `G` + `H`             | 回到首頁                 |
| `G` + `K`             | 前往假名學習             |
| `G` + `W`             | 前往單詞學習             |
| `G` + `P`             | 前往句子學習             |
| `Enter`               | 開始學習                 |
| `Space` / `N` / `→`   | 下一個                   |
| `H`                   | 顯示/隱藏提示            |
| `P` / `V`             | 播放發音                 |

## 📖 使用說明

### 假名學習

1. 點擊「設置」選擇要練習的假名類型（清音、濁音、拗音）
2. 選擇顯示模式和練習模式
3. 點擊「開始」進入練習
4. 使用鍵盤快捷鍵快速操作

### 單詞學習

1. 在設置中選擇假名範圍
2. 系統自動篩選包含所選假名的單詞
3. 學習模式下自動顯示中文和發音
4. 記憶模式下手動控制提示

### 句子學習

1. 選擇學習場景（日常對話、旅遊、商務等）
2. 可多選場景組合學習
3. 支持顯示中文提示和日語發音

## 💡 學習建議

- **初學者**: 從假名學習開始 → 清音學習模式 + 自動發音
- **進階者**: 單詞學習 → 按假名行逐步擴展詞彙量
- **熟練者**: 句子學習 → 場景化練習，提升實戰能力
- **記憶強化**: 使用記憶模式 + 互換/羅馬音模式反向練習

## 🛠️ 技術棧

- **框架**: Vite + React 18
- **路由**: React Router v6
- **UI**: Tailwind CSS + shadcn/ui
- **語音**: Web Speech API
- **語言**: TypeScript
- **狀態管理**: React Hooks + Local Storage
- **部署**: Vercel

## 📊 數據來源

- **假名數據**: 包含 104 個假名的完整五十音圖
- **單詞數據**: 精選日語常用單詞，標註音調和中文釋義
- **句子數據**: 場景化常用句子，涵蓋多種實用場景
