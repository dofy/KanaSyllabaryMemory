# 假名記憶 - Kana Syllabary Memory

一個優雅的日語假名（平假名、片假名）記憶練習 Web 應用

## ✨ 功能特性

### 🎯 核心功能

- **全面的假名支持**

  - 清音（46 個基礎假名）
  - 濁音（25 個濁化假名）
  - 拗音（33 個拗音假名）
  - 共計 104 個假名

- **多種顯示模式**

  - 🔀 **混合模式** - 隨機顯示平假名或片假名
  - 🅰️ **平假名模式** - 專注學習平假名
  - 🅱️ **片假名模式** - 專注學習片假名
  - 🔤 **羅馬音模式** - 通過羅馬音反向學習
  - ⇄ **互換模式** - 平假名與片假名互相轉換

- **雙重練習模式**

  - 📚 **學習模式** - 自動顯示提示，適合初學者
  - 🧠 **記憶模式** - 手動顯示提示，適合複習鞏固

- **智能學習系統**
  - 隨機抽取避免連續重複
  - 自動循環練習
  - 智能提示系統（根據顯示內容自動匹配提示信息）
  - 標準日語發音播放

### 🎨 用戶體驗

- **主題切換**

  - 亮色模式（適合白天）
  - 暗色模式（護眼模式）
  - 支持系統主題自動切換

- **響應式設計**

  - 完美適配手機、平板、桌面
  - 移動端全屏 App 風格佈局
  - 觸摸友好的界面設計

- **數據持久化**
  - 實時自動保存所有設置
  - 記住用戶選擇的假名
  - 保存顯示模式和練習模式偏好

### ⚙️ 靈活配置

- **自定義選擇**

  - 按類型批量選擇（清音/濁音/拗音）
  - 單個假名精確控制
  - 分組顯示，清晰易用

- **統計信息**
  - 實時顯示已選假名數量
  - 分類統計清音/濁音/拗音
  - 學習進度一目了然

## 🚀 快速开始

### 本地运行

```bash
# 克隆项目
git clone https://github.com/dofy/KanaSyllabaryMemory.git
cd KanaSyllabaryMemory

# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 浏览器访问 http://localhost:3000
```

### 构建部署

```bash
# 构建生产版本
pnpm build

# 静态导出（用于 GitHub Pages）
pnpm export
```

## 📖 使用指南

### 基本使用

1. **開始練習**

   - 點擊「開始」按鈕開始學習
   - 點擊假名右側的音量圖標播放發音
   - 點擊「顯示提示」查看答案（記憶模式）
   - 點擊「下一個」繼續練習

2. **配置學習內容**

   - 點擊「設置」按鈕打開配置面板
   - 選擇練習模式（學習模式/記憶模式）
   - 選擇顯示模式（混合/平假名/片假名等）
   - 選擇要學習的假名類型
   - 所有設置實時自動保存

3. **主題切換**
   - 點擊右上角月亮/太陽圖標
   - 切換亮色/暗色主題
   - 設置自動保存

## 🎓 學習資源

### 假名表

本應用包含完整的日語假名：

- **清音（46 個）**: あいうえお、かきくけこ...
- **濁音（25 個）**: がぎぐげご、ざじずぜぞ...
- **拗音（33 個）**: きゃきゅきょ、しゃしゅしょ...

### 學習建議

1. **初學者**: 建議選擇「學習模式」，從清音開始，熟練後再學習濁音和拗音
2. **進階學習**: 使用「記憶模式」配合「互換模式」加強平假名和片假名的對應關係
3. **反復練習**: 利用「羅馬音模式」進行反向學習，加深記憶
